Easy mathematics

I've noticed that more and more CTFs include math tests.
So I'm developing an API to solve some of these problems.
If you'd like to contribute and add new calculs, the code is available here : here

API Routes Documentation

  • /api/add - POST

    Adds two numbers. Parameters: a, b. Returns the sum.

  • /api/sub - POST

    Subtracts two numbers. Parameters: a, b. Returns the difference.

  • /api/mul - POST

    Multiplies two numbers. Parameters: a, b. Returns the product.

  • /api/eq - POST

    Checks equality of two numbers. Parameters: a, b. Returns true if equal, false otherwise.

  • /api/polynome - POST

    Solves a quadratic equation. Parameters: a, b, c. Returns the roots if real, error message otherwise.
